01. Proyecto: Buscador de Wikipedia

Se trata de una copia del buscador de Wikipedia con el cual, al escribir el nombre del elemento sobre el cual se pretende encontrar información, la aplicación realiza una petición HTTP a la API de Wikipedia y muestra los resultados. Se muestran varios de los resultados de la búsqueda con su título y descripción y, haciendo click sobre el elemento de interés se accede a su artículo de Wikipedia.

02. Proyecto: Aplicación del tiempo

En este proyecto se ha creado una aplicación del tiempo. Implementando dos peticiones HTTP a una API libre del tiempo, una vez el usuario ha escrito el nombre de la ciudad sobre la que desea conocer el tiempo, entonces la aplicación muestra la información del tiempo de dicha ciudad. Además, el usuario tiene un tiempo para introducir el nombre sin que la aplicación realice una búsqueda en caso de que se equivoque con el nombre.
